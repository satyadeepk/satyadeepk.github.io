Place the following in httpd.conf or apache.conf in your apache installation folder.

<Directory /var/www/>

# enable ssi directives
Options +Includes

AddType text/html .shtml
AddOutputFilter INCLUDES .shtml

#Â  avoid duplicated <html><body> and no column sorting

# place folders first
# IndexOptions FoldersFirst

# define absolute path to the header file from server home
HeaderName /header.html

ReadmeName /readme.html

</Directory>

then restart apache.
