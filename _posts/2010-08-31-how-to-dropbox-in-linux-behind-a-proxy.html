If you do not know what Dropbox is,  it is a popular cross-platform file synchronization and backup application. It syncs the Dropbox folder present in your computer to the cloud by integrating itself with the file manager of the operating system.

In case of Ubuntu, it depends upon the default file manager called "nautilus" of the GNOME Desktop to provide the user interface like showing the overlay icons over the files. The following are the steps to install Dropbox, if you are behind a proxy or a firewall.

  1. Download the nautilus-dropbox package from dropbox website https://www.dropbox.com/downloading?os=lnx . When you install it, it first installs the dropbox plugin for nautilus and then tries to fetch the Dropbox daemon from the internet. But if you are a behind a proxy, this connection fails and reports an error.
  2. Hence download the actual Dropbox daemon manually using these links 32 Bit (x86),  64 Bit( x86_64 ). If you do not know which one your system is , check your kernel type with "uname -m" in terminal, it'll say "x86_64" if it's a 64 bit kernel.
  3. Place the .tar.gz file in your home directory.

Change to your home directory /home/satyadeep for example and Run the following in terminal to extract the archive:

[bash] tar xzf dropbox* [/bash]

This will create a hidden directory called .dropbox-dist in your home folder.

  1. Start Dropbox by executing the following command.
    [bash] ~/.dropbox-dist/dropboxd[/bash]

In case, if there is a newer version available, or you want to try out a testing build from the Dropbox forums, simply extract the latest .tar.gz file into your home folder as above (Step 3) and restart the dropbox daemon. That will upgrade you to the latest version.
